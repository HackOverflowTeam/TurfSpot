üêõ Troubleshooting

If you're being logged out automatically:

  1. Open browser console (F12)
  2. Check if there are any errors
  3. Clear localStorage: Run this in console: 
    localStorage.clear()
  4. Login again from index.html
  5. Check your token: 
    console.log(localStorage.getItem('token'))

If backend is not responding:

  1. Check if it's running: Look for "Server running on port 4000" message
  2. Test the health endpoint: 
    curl http://localhost:4000/api/health
  3. Check for port conflicts: 
    lsof -i :4000

üß™ Testing Flow

Test as Admin:
